Build a TIC-TAC-TOE Web Application using html css and javascript during my Web Development Internship at SkillCraft Technology.
I built this project to practice front-end logic, UI design, and clean JavaScript.
🔧 Key Functions:
Player Turn System – Automatically switches between X and O after every move.
Win Detection – Checks all winning combinations (rows, columns, diagonals) in real-time.
Draw Detection – Declares a tie when all boxes are filled with no winner.
Reset Function – Instantly restarts the match without refreshing the page.
Game Locking – Board becomes inactive once a winner or draw is detected.
Responsive UI – Works smoothly on both mobile and desktop.
